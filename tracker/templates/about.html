🚆 About Train Tracker

Train Tracker is a web application that helps users search Indian railway trains, view train information, check the complete route, and manage favorite trains and search history through a simple and user-friendly interface.


🎯 Project Features

  • 🔍 Search trains by Name or Number
  • 🚆 View Train Information
  • 🚉 Display Complete Train Route
  • ⭐ Add and Remove Favorite Trains
  • 📜 Automatically Save Search History
  • 🗑 Delete Individual History
  • 🧹 Clear All Search History
  • 📱 Responsive User Interface

💻 Technologies Used

Frontend HTML, CSS, JavaScript
Backend Python Django
Database SQLite
API RailRadar API

🚀 Project Workflow

User searches for a train by entering its name or number. The application retrieves train information from the RailRadar API, displays train details and the complete route, and allows users to save favorite trains and maintain search history using the SQLite database.


🎓 Purpose of the Project

This project was developed to learn Full Stack Web Development using Django and to demonstrate API integration, database operations, and responsive web application development.
